# PPEL-Lab 웹사이트 레이아웃 전면 개편

## 작업 일시
2025년 12월 23일

## 개편 목표
라이트 모드를 기본으로 설정하고, 과도한 여백을 제거하며, 작은 카드 형태로 콤팩트하고 가독성 있는 레이아웃으로 전면 개편합니다.

---

## 주요 변경사항

### 1. 라이트 모드를 기본으로 설정

웹사이트의 기본 테마를 **다크 모드에서 라이트 모드로 변경**하여 전문적이고 깔끔한 느낌을 제공합니다.

#### 변경 내용
- **기본 배경색**: 검은색 (#0a0a0a) → **흰색 (#ffffff)**
- **카드 배경**: 반투명 검은색 → **순백색 (#ffffff)**
- **텍스트 색상**: 흰색 → **검은색 (#1a1a1a)**
- **테두리 색상**: 반투명 흰색 → **회색 (#e0e0e0)**
- **네비게이션**: 검은 반투명 → **흰색 반투명 (0.9 opacity)**

#### 다크 모드 지원
- 사용자가 원할 경우 테마 토글 버튼(🌙/☀️)으로 다크 모드 전환 가능
- 다크 모드 클래스를 `light-mode`에서 `dark-mode`로 변경

---

### 2. 섹션 여백 최소화 및 컴팩트 레이아웃

과도한 여백을 제거하고 콘텐츠를 효율적으로 배치하여 한 화면에 더 많은 정보를 표시합니다.

#### 섹션 패딩 축소
- **변경 전**: 150px 80px
- **변경 후**: 60px 30px
- **감소율**: 60% (상하), 62.5% (좌우)

#### 카드 간격 최적화
- **멤버 카드 간격**: 30px → **20px**
- **논문 카드 간격**: 35px → **20px**
- **통계 카드 간격**: 유지 (30px)

---

### 3. 작은 카드 형태로 통일

모든 카드를 작고 깔끔한 형태로 통일하여 일관성 있는 디자인을 구현합니다.

#### 멤버 카드 개선
- **그리드 최소 너비**: 350px → **200px** (43% 축소)
- **카드 높이**: 280px (고정) → **auto** (콘텐츠에 맞춤)
- **카드 패딩**: 40px 30px → **25px 20px**
- **아바타 크기**: 120px → **100px**
- **Border Radius**: 20px → **12px**
- **한 줄에 표시**: 3-4개 → **5-6개** (화면 크기에 따라)

#### 논문 카드 개선
- **카드 패딩**: 40px → **25px**
- **카드 간격**: 35px → **20px**
- **Border Radius**: 25px → **12px**
- **테두리**: 2px 투명 → **1px 회색**

#### 통계 카드 개선
- **카드 패딩**: 50px 40px → **30px 25px**
- **Border Radius**: 25px → **12px**

#### Standard Card 통일
모든 `standard-card` 클래스를 사용하는 요소에 일관된 스타일 적용:
- **Border Radius**: 20px → **12px**
- **Padding**: 35px → **25px 20px**
- **Shadow**: 새로운 `--card-shadow` 변수 사용
- **Hover 효과**: translateY(-5px) → **translateY(-3px)**

---

### 4. 글자 크기 최적화

가독성을 유지하면서 콤팩트한 레이아웃에 맞게 폰트 크기를 조정합니다.

#### 기본 타이포그래피
- **Body 폰트**: 18px → **16px**
- **Line Height**: 1.8 → **1.7**
- **섹션 타이틀**: 3.5rem → **2.8rem**

#### 논문 섹션
- **논문 제목**: 1.4rem → **1.3rem**
- **저자명**: 1.1rem → **1.05rem**
- **저널명**: 1.1rem → **1.05rem**

#### 멤버 섹션
- **멤버 이름**: **1.2rem** (신규 정의)
- **멤버 역할**: **1.0rem** (신규 정의)

---

### 5. 새로운 디자인 시스템 변수

일관된 디자인을 위한 CSS 변수 추가:

```css
--card-shadow: 0 2px 8px rgba(0, 0, 0, 0.08);
```

모든 카드에 통일된 그림자 효과를 적용하여 깔끔한 느낌을 제공합니다.

## 사용자 경험 개선

### 가독성
- 라이트 모드로 눈의 피로 감소
- 적절한 폰트 크기로 읽기 편의성 유지
- 명확한 카드 구분으로 정보 구조화

### 정보 밀도
- 한 화면에 더 많은 정보 표시
- 스크롤 횟수 감소
- 효율적인 공간 활용

### 디자인 일관성
- 모든 카드에 동일한 Border Radius (12px)
- 통일된 그림자 효과
- 일관된 패딩 및 간격

### 전문성
- 깔끔한 흰색 배경
- 정돈된 레이아웃
- 학술 웹사이트에 적합한 디자인
