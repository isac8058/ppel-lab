# 광주채플 식당봉사 및 부목자모임 운영 컨텍스트 (2026 하반기 갱신본)

본 문서는 2026년도 하반기 목장 편성표(부부목장, 여자목장)를 반영한 갱신본으로, 2026-06 작성 컨텍스트를 대체합니다. 작성 주체는 임수만 부목자(A조 정조장, 215 김영석A 마을)입니다.

## 0. 공통 작성 규칙

- 외부 공지 호칭은 광주채플로 통일합니다. 한인구 초원님 호칭은 유지합니다.
- 부목자 개인은 이름 뒤에 부목자님을 붙여 호칭합니다.
- 인사말은 샬롬으로 시작하는 형식을 기본으로 합니다.
- 동명이인 구분용 영문 접미(A, B, C 등)는 편성표 표기 그대로 유지합니다(예: 김준영H, 정재성A).

## 2. 식당봉사 (식당 홀 청소)

### 2.1 운영 방식 (변경 없음)

- 2조(A조/B조) 격주 순환. 격주 교대 기준점은 2026-05-03(일)=A조이며, 이후 14일 간격으로 A, B가 교대합니다.
- 차례 판정 로직: (해당 일요일 날짜 빼기 2026-05-03)을 일수로 환산한 뒤 7로 나눈 몫을 2로 나눈 나머지가 0이면 A조, 1이면 B조입니다.
- 검증 차례(하반기): 07/12=A조, 07/19=B조, 07/26=A조, 08/02=B조.
- 알림 발송: 매주 금요일 09:30 KST(00:30 UTC), GitHub Actions(ppel-lab/volunteer-reminder) 자동 발송. 수신 주소 smlim@jbnu.ac.kr.

### 2.2 청소 제외 규칙 (명단 갱신)

- SG목장 부목자 3명(211 정상연 소속: 김현호A, 김진남, 이정일)은 식당봉사 자체에서 제외됩니다(편성 미포함).
- 주일목장 부목자 2명(임철승, 김승호D)은 식사 직후 목장 모임을 인도해야 하므로 식당 홀 청소만 제외됩니다. 식당봉사 자체에는 참여합니다.
- 김종민, 강신우는 부목자 신분이 아니므로 하반기부터 제외 명단에서 삭제합니다.

- 편성 원칙: 기존 조 소속을 유지하고 신규 3명만 결원 조에 배정했습니다. 모종명을 A조에 배치하면 A조가 전 마을 2명씩(211~215 각 2명)으로 균형을 이루므로, 나머지 신규 2명(박철수B, 안정욱D)은 B조에 배정했습니다.

### 2.4 특이사항

- 홍성일A 부목자는 가족 간병으로 현장 예배 참석이 어려웠고 연락 두절 이력이 있습니다(2026-05). 편성에는 유지하되 실참여는 별도 관리합니다.
- 김대연A 부목자는 주일 격주 근무로 부담을 표한 이력이 있으나, 직전 확정본 기준 B조 부조장직을 유지합니다.
- 편성표 표기 장페남은 기존 명단 기준 장쾌남으로 처리했습니다(인쇄 표기 확인 필요).

### 2.5 식당봉사 알림 메시지 형식 (2026-07 변경: 직책 미표기, 이름만 나열)

정조장, 부조장 직책 표기를 제거하고 전원 이름만 나열합니다. 직책은 내부 관리용으로만 유지합니다.

```
[식당봉사 알림] 07/19

이번주 식당봉사는 B조입니다.

정정진, 김대연A, 김도윤B, 박철수B, 오선록, 김중백, 김경은E, 정준기C, 정재성A, 안정욱D

— 광주채플
```

A조 차례에는 조 이름과 명단만 교체합니다(임수만, 최성계, 김호기, 모종명, 김준영H, 김병수C, 박광진, 홍성일A, 장쾌남, 김성철E).
